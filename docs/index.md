A personal AI orchestrator (Linux only). Built to keep the human from becoming the bottleneck when several AI coding agents run in parallel: agents are given room to run autonomously, a sandbox with a configurable write scope keeps them from doing damage, and every in-flight task is visible at a glance through a Web UI and TUI — all on your own machine.
